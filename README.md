# Student Grade Calculator

This is a basic web application that calculates a student's total marks, average, grade, and pass/fail result.

## Features
- Enter marks for 5 subjects
- Calculates total and average
- Displays grade (A+ to F)
- Shows PASS or FAIL

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or copy the code
2. Save it as `index.html`
3. Open the file in any web browser

## Example Output
Total: 433  
Average: 86.60  
Grade: A  
Result: PASS
